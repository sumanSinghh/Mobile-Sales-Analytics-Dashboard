# Mobile-Sales-Analytics-Dashboard
📊 Mobile Sales Analytics Dashboard built using Power BI to analyze mobile sales performance. The dashboard provides interactive insights into sales trends, revenue, units sold, top-performing brands and models, regional performance, and customer purchasing patterns using dynamic charts, filters, and KPIs for effective business decision-making.


Overview

This project is an interactive Power BI dashboard designed to analyze mobile sales data and provide meaningful business insights. It helps users monitor sales performance, identify top-performing products, and make data-driven decisions.

Features

- 📈 Sales trend analysis
- 💰 Revenue and profit KPIs
- 📱 Brand and model-wise sales performance
- 🌍 Region-wise sales analysis
- 👥 Customer purchasing insights
- 🎛️ Interactive filters and slicers for dynamic reporting

Tools & Technologies

- Power BI
- Microsoft Excel
- Data Cleaning
- Data Modeling
- DAX
- Data Visualization
